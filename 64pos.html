64POS UX Internship

Project Background

Over the summer, I was referred by my friend to a small start-up, 64 POS, to work as an UX intern. Though I joined late and had only worked for about a month, I was able to get some hands-on experience with working on user experience concepts for real users. The particular project I worked on was developing a web aggregator mobile web-app.

Team and your Role

I was taken in as a UX intern where I had the opportunity to develop analysis and technical skills necessary to fully understand and develop for users. My team had a few other UX interns, software developers, and a project manager to help manage the process.

The Work

Some of the work I was given inlcuded:
  • Performing competitor analysis research and point of view analysis to find user needs. Used several news apps to find what UI/UX functionalities were common and distinguished from there which ones were necessary.
  • Creating wireframe mockups for some screens based on workflow chart. The UI was based on several other apps and what features were deemed important in our team.

Results

I learned more about the process behind user experience and how teams of designers work with each other. The experience, though brief, had value in learning more about the workings of a designer.
